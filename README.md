Style and standards for: ts, js, vue, yml, json

## Usage:
```sh
npm i -D @app-research/eslint-config

# peerDeps (auto installed for npm >v7)
npm i -D eslint prettier
```

Create `.eslintrc` in your project root
```js
{
    extends: '@app-research'
}
```
